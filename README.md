# Spy Game

Мобильное веб-приложение для игры "Шпион".

## Как играть

- Задайте количество игроков.
- Выберите режим таймера и время, если хотите сыграть на время.
- Нажимайте на карточку по очереди: первое нажатие показывает слово или роль, второе скрывает карту и передает ход следующему игроку.
- Если игроку выпадает роль шпиона, он должен притвориться, что знает слово.
- Если время истекает до того, как игроки определят шпиона, побеждает шпион.
- В любой момент можно остановить таймер и завершить игру.

## Возможности

- Интуитивный мобильный интерфейс.
- Случайное слово или собственный список слов.
- Автоматическое распределение одной или нескольких ролей-шпионов.
- Таймер с возможностью остановки.
- Кнопка завершения игры в любой момент.

## Запуск

1. Откройте `index.html` в браузере.
2. Настройте количество игроков и слова.
3. Нажмите "Начать игру".
